BindObject

UMLClass
UCR :: Class Diagram :: BindObject

Description

Describes a binding (eg CTRL+ALT+Left Mouse) plus the various possible options (eg wild, block).

Attributes

Visibility Name Type Description
public Keys Key

The list of keys that comprise the binding. Indexed array, ordered by the order the keys were pressed.
public Wild Bool

Analogous to AHK's * parameter for hotkeys. eg Makes "a" trigger even if CTRL is held.
public Block Bool

Whether or not to block the input from being seen by other applications. Block off is synonymous with the ~ prefix in AHK Called "Block" rather than "Passthrough" for UX reasons.
public SuppressRepeats Bool

Keyboard keys, by default, repeat when held. For some use-cases, this may be undesirable behavior for the end-user. This option allows them to not repeatedly fire the ChangeState callback if the key is held.
