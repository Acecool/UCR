BindModeHandler

UMLClass
UCR :: Class Diagram :: BindModeHandler

Description

Handles user selection of Hotkeys. Presents "Press desired key combination" dialog and waits for user input. When user input complete, fires a callback.

Operations

Visibility Name Description
public StartBindMode(Hotkey, Callback)

Starts off the process of allowing the End-User to select a hotkey. Pass the Hotkey object that is being bound
public SetHotkeyState(bool)

Turns on or off hotkeys for every key on the keyboard, to facilitate detecting of user input. When binding hotkeys, it pre-binds a key object to the up and down event of every key.
public ProcessInput(i: Key, event: bool)

When a key or button is pressed in bind mode, this gets called with a pre-bound Key object and the event (up or down) that occurred,
